# Lazy Chef
Lazy Chef is a client-side AI recipe generator that turns available ingredients into structured, realistic recipes using the Gemini API.

It supports image-based ingredient detection, multilingual output, and customizable cooking time preferences.

Currently in development. Available at https://lazy-chef-psi.vercel.app/

---

## Why use it? 
Cook faster and easier without sacrificing quality.

## Run Locally

### 1. Clone the repository
```bash
git clone https://github.com/RomanMakarov6666/lazy-chef.git
```
2. Navigate to the project directory
```bash
cd lazy-chef
```
3. Install dependencies
```bash
npm install
```
4. Start server
```bash
npm run dev
```
5. Open in browser
```bash
http://localhost:5173
```
